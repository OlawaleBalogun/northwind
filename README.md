# Northwind Traders Retail Analytics Project

## Project Overview
This project is a complete retail and supply chain analytics solution built using the **Northwind Traders** dataset. The project combines **MySQL** for data analysis and **Power BI** for interactive dashboard reporting.

The objective of this project was to analyze business performance across sales, customers, employees, logistics, inventory, and geographic markets while answering real-world business questions using SQL and data visualization techniques.

---

- MySQL Workbench
- Power BI Desktop
- SQL (MySQL)

- Northwind Dataset

## Business Insights Discovered
- Revenue is highly concentrated among a small number of customers and product categories.
- Seasonal sales trends significantly influence monthly revenue performance.
- Delivery delays impact operational efficiency and customer satisfaction.
- Inventory management issues create risks of stock shortages and overstocking.
- Employee productivity varies significantly across teams.
